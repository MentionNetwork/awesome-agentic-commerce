# Shopify — Top Articles & Guides

The best articles, tutorials and engineering write-ups for building on and selling with Shopify — for developers and store owners alike.

## For Developers

- [Build a Shopify App Using React Router](https://shopify.dev/docs/apps/build/build?framework=remix) - Shopify. Official hands-on tutorial building a QR-code app with App Bridge, metaobjects and the GraphQL Admin API — the recommended modern app path.
- [How to Build a Shopify App: The Complete Guide](https://www.shopify.com/partners/blog/how-to-build-a-shopify-app) - Shopify Partners, 2021. Strategy-level walkthrough of the six stages from idea validation to App Store submission.
- [Create a Theme](https://shopify.dev/docs/storefronts/themes/getting-started/create) - Shopify. Official getting-started guide for the theme dev workflow using Shopify CLI and the Skeleton reference theme.
- [Liquid Storefronts for Theme Developers](https://www.shopifyacademy.com/path/liquid-storefronts-for-theme-developers) - Shopify Academy. Three-course path covering Liquid basics, theme architecture and performance optimization.
- [Hydrogen: Shopify's Headless Commerce Framework](https://hydrogen.shopify.dev/) - Shopify. Canonical reference for the Remix-based headless framework with commerce components and free Oxygen edge hosting.

## Engineering Deep-dives

- [The Generative Recommender Behind Shopify's Commerce Engine](https://shopify.engineering/generative-recommendations) - Shopify Engineering, 2026. Sequence-based generative recommendation model that lifted Shop orders and click-through rate.
- [How We Prepare Shopify for BFCM](https://shopify.engineering/bfcm-readiness-2025) - Shopify Engineering, 2025. Nine-month readiness program (capacity planning, chaos engineering, load testing) that sustained 200M+ requests/minute.
- [Capacity Planning at Scale](https://shopify.engineering/capacity-planning-shopify) - Shopify Engineering, 2020. Evergreen deep-dive on forecasting cloud resources and load-testing for extreme peak traffic.

## For Store Owners

- [How To Start an Online Store in 2026](https://www.shopify.com/blog/start-online-store) - Shopify, 2026. Merchant-focused 10-step launch guide from product sourcing to marketing.
- [Conversion Rate Optimization (CRO): Get Started](https://www.shopify.com/blog/120261189-conversion-rate-optimization) - Shopify, 2026. Primer on CRO fundamentals and strategies to turn visitors into buyers.
