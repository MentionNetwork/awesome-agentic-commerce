# Contribution Guidelines

Thanks for taking the time to contribute! Please make sure your suggestion is genuinely *awesome* — a resource you would recommend to a friend building or running an online store.

## Adding to the list

1. Fork the repository and create a new branch.
2. Add your entry to the most relevant section, keeping the list **alphabetically ordered** within that section.
3. Use this exact format:

   ```
   - [Name](https://example.com) - Short description ending with a period.
   ```

4. Open a pull request with a clear title and a one-line reason the resource belongs here.

## Entry rules

- The link must point to the canonical homepage or repository, not a tracking or affiliate URL.
- Descriptions must be concise, factual, and end with a period.
- Do **not** start a description with "A", "An", or "The".
- No duplicates — check the list first.
- English only.

## Quality bar

- The resource must be actively maintained and genuinely useful to store owners, developers, agencies, or marketers.
- Self-promotion is allowed only for resources that are clearly best-in-class for their category.
- Broken or abandoned links may be removed.

## Reporting issues

Found a dead link, a mislabeled entry, or a better alternative? Open an issue — housekeeping PRs are just as welcome as new entries.
